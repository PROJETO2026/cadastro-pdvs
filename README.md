# Cadastro de PDVs

Sistema Laravel para cadastro de Pontos de Venda.

## Funcionalidades

- Cadastro de PDV
- Cadastro de responsável
- Cadastro de cardápio
- Listagem de produtos

## Tecnologias

- PHP
- Laravel
- Blade
- MySQL

## Instalação

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
